# Development Getting Started

## Project Overview

The main control chip of the Starry Sky C2 board utilizes the team's self-developed retroSoC. It is an open-source ASIC customization framework that supports integrating different types of open-source RISC-V processor cores (self-developed or third-party) and facilitates tape-out. The current version of retroSoC integrates the renowned open-source RISC-V processor core PicoRV32. It fully implements the RV32IMC instruction set and can be flexibly configured into various combinations such as RV32E, RV32I, RV32IC, RV32IM, and RV32IMC. Furthermore, retroSoC also includes numerous open-source peripheral modules, such as UART, SPI, I2C, GPIO, PWM, etc., significantly enhancing the board's demonstration and application potential.



!!! warning "注意事项"
    Overall, the functionality and performance of the board's main control chip are roughly equivalent to ST's F1 mid-to-low-end series. However, due to potential software or hardware errors, this board is currently intended for embedded prototyping use only (e.g., internal testing, participation in competitions). It must not be used in any actual production or operational environments.



## Chip Introduction

### Hardware Architecture

| Feature | Description  |
| :--- | :--- |
| **Processor Core** | PicoRV32 |
| **Instruction Set Architecture** | RISC-V rv32imac |
| **Operating Frequency** | 72MHz |

### Memory System

| Type  | Capacity | Address Range |
| :--- | :--- | :--- |
| **On-chip SRAM** | 128KB | `0x30000000` - `0x3001FFFF` |
| **External PSRAM** | 8MB | `0x40000000` - `0x407FFFFF` |
| **SPI Flash** | 16MB | `0x00000000` - `0x00FFFFFF` |

### Peripheral Modules

The project includes a rich set of peripheral driver modules:

#### Communication Interfaces

*   **SYS_UART**：Standard serial communication
*   **HP_UART**：High-performance serial communication
*   **SPI**：Standard SPI interface
*   **I2C**：I2C bus interface
*   **PS2**：PS2 keyboard/mouse interface

#### Storage and Control

*   **GPIO**：16 General Purpose Input/Output
*   **PWM**：4 Pulse Width Modulation outputs
*   **Timer**：2 Timers
*   **Flash**：SPI Flash controller
*   **PSRAM**：External PSRAM controller

#### Others

*   **RNG**：Random Number Generator

*   **ArchInfo**：System architecture information

	

## Board Introduction

![board](../../../res/img/brd/starry-sky-c/v2.0/board1.webp)

Introduction to the Starry Sky C2 Pi peripheral interfaces:

*   **System**：
    *   1 Power switch, red power indicator LED
    *   1 Type-C power interface (also serves as the SoC's UART debug interface)
    *   1 Drag-and-drop programmer: HFP-LINK
*   **SoC**：
    *   1 72MHz active crystal oscillator
    *   1 Active crystal oscillator pad position
    *   1 UART debug interface, using CP2102 for protocol conversion
    *   1 Reset button
    *   1 8MB PSRAM
    *   1 16MB SPI Flash
    *   1 PS2 interface
    *   1 SPI color screen
    *   2 Sets of fanned-out IO pins
