# Templates 模块与板卡支持情况说明

!!! info "文档说明"

    本文档用于说明 embedded-sdk 工程 `templates` 目录下各模板模块的功能用途，以及不同板卡当前支持的模板范围。



## 各模块功能介绍

此表格是按照 `templates` 目录下的模块名称，对每个模板的主要功能进行说明。

| 序号 | 模块名称           | 功能介绍                                                     |
| ---: | ------------------ | ------------------------------------------------------------ |
|    1 | archinfo         | 架构信息测试模板，主要用于读取或打印当前 soc / cpu 架构相关信息，验证基础运行环境是否正常。 |
|    2 | asm_gpio         | 汇编语言 GPIO 控制示例，用于验证在汇编程序中直接访问 GPIO 外设的能力。 |
|    3 | asm_hello        | 汇编语言基础输出示例，通常用于验证启动流程、串口输出以及汇编工程的基本运行环境。 |
|    4 | crc              | CRC 校验功能测试模板，用于验证片上 CRC 模块或软件 CRC 计算功能是否正常。 |
|    5 | filesystem_test  | 文件系统测试模板，用于验证文件系统挂载、文件读写等基础功能。 |
|    6 | gpio             | GPIO 通用测试模板，用于验证 GPIO 输入输出控制能力，如点灯、按键检测或普通 IO 翻转。 |
|    7 | hello            | 最基础的 Hello 示例模板，用于验证工程编译、下载、启动和串口打印流程。 |
|    8 | i2c_scan         | I2C 总线扫描模板，用于扫描 I2C 总线上挂载的设备地址，验证 I2C 通信是否正常。 |
|    9 | i2c_sgp30        | SGP30 气体传感器测试模板，通过 I2C 总线读取 SGP30 传感器数据。 |
|   10 | minesweeper      | 扫雷小游戏应用模板，属于综合应用示例，可用于验证显示、输入和基础逻辑运行能力。 |
|   11 | ps2              | PS2 接口测试模板，用于验证 PS2 键盘或相关外设输入。          |
|   12 | pwm              | PWM 输出测试模板，用于验证脉宽调制输出功能，可用于蜂鸣器、背光调节、电机控制等场景。 |
|   13 | qspi             | QSPI 接口测试模板，用于验证 QSPI Flash 或相关高速串行存储接口访问功能。 |
|   14 | rcu              | 复位与时钟控制相关测试模板，用于验证系统时钟、复位控制或外设时钟使能等功能。 |
|   15 | rng              | 随机数发生器测试模板，用于验证硬件 RNG 或随机数生成相关功能。 |
|   16 | rtc              | 实时时钟测试模板，用于验证 RTC 时间计数、时间读取或定时唤醒等功能。 |
|   17 | shell_test       | Shell 功能测试模板，用于验证命令行交互、串口输入输出和基础命令解析功能。 |
|   18 | smoke_test       | 冒烟测试模板，用于快速检查系统基础功能是否正常。             |
|   19 | spi_flash_test   | SPI Flash 测试模板，用于验证 SPI Flash 的读、写、擦除等基本访问功能。 |
|   20 | spi_st7735       | ST7735 SPI 屏幕显示模板，用于验证 ST7735 显示屏初始化和图像显示功能。 |
|   21 | spi_st7735_clock | 基于 ST7735 屏幕的时钟显示示例，用于验证 SPI 屏幕显示与时间显示逻辑。 |
|   22 | spi_st7735_dout  | ST7735 SPI 数据输出测试模板，用于验证向 ST7735 屏幕发送显示数据的功能。 |
|   23 | st7789           | ST7789 屏幕显示模板，用于验证 ST7789 屏幕驱动、初始化和显示输出功能。 |
|   24 | timer            | 定时器测试模板，用于验证定时中断、周期计数或延时功能。       |
|   25 | wdg              | 看门狗测试模板，用于验证看门狗启动、喂狗以及系统复位保护功能。 |



## 板卡支持情况

### C1 板卡支持的模板

- `asm_hello`：汇编基础输出示例
- `gpio`：GPIO 通用测试示例
- `hello`：基础 Hello 测试示例
- `i2c_scan`：I2C 总线扫描示例
- `minesweeper`：扫雷小游戏应用示例
- `spi_st7735_dout`：ST7735 SPI 数据输出示例

!!! info "说明"

    `c1` 当前主要覆盖基础启动、GPIO、I2C 以及部分显示应用示例，适合作为基础功能验证板卡。



### C2 板卡支持的模板

- `asm_hello`：汇编基础输出示例
- `filesystem_test`：文件系统测试示例
- `gpio`：GPIO 通用测试示例
- `hello`：基础 Hello 测试示例
- `i2c_scan`：I2C 总线扫描示例
- `minesweeper`：扫雷小游戏应用示例
- `shell_test`：Shell 命令行测试示例
- `spi_flash_test`：SPI Flash 测试示例
- `spi_st7735_dout`：ST7735 SPI 数据输出示例
- `st7789`：ST7789 屏幕显示示例

!!! info "说明"

    `c2` 相比 `c1` 支持的模板数量更多，除了基础外设外，还包含文件系统、Shell、SPI Flash 和屏幕显示相关测试，适合进行更完整的外设功能验证。



### L3 板卡支持的模板

- `asm_gpio`：汇编 GPIO 控制示例
- `asm_hello`：汇编基础输出示例
- `gpio`：GPIO 通用测试示例
- `hello`：基础 Hello 测试示例
- `i2c_scan`：I2C 总线扫描示例

!!! info "说明"

    `l3` 当前主要覆盖基础启动、汇编示例、GPIO 和 I2C 扫描，适合用于基础工程移植和低层外设访问验证。



### L3_1 板卡支持的模板

- `archinfo`：架构信息测试示例
- `asm_gpio`：汇编 GPIO 控制示例
- `asm_hello`：汇编基础输出示例
- `crc`：CRC 校验测试示例
- `gpio`：GPIO 通用测试示例
- `hello`：基础 Hello 测试示例
- `i2c_scan`：I2C 总线扫描示例
- `i2c_sgp30`：SGP30 传感器测试示例
- `minesweeper`：扫雷小游戏应用示例
- `ps2`：PS2 接口测试示例
- `pwm`：PWM 输出测试示例
- `qspi`：QSPI 接口测试示例
- `rcu`：复位与时钟控制测试示例
- `rng`：随机数发生器测试示例
- `rtc`：实时时钟测试示例
- `smoke_test`：冒烟测试示例
- `spi_st7735_clock`：ST7735 时钟显示示例
- `spi_st7735_dout`：ST7735 SPI 数据输出示例
- `st7789`：ST7789 屏幕显示示例
- `timer`：定时器测试示例
- `wdg`：看门狗测试示例

!!! info "说明"

    `l3_1` 是当前截图中模板支持数量最多的板卡，覆盖了基础启动、汇编示例、GPIO、I2C、SPI 显示、QSPI、PWM、定时器、RTC、看门狗、CRC、RNG、RCU 等多个方向，适合作为主要功能验证和综合测试平台。



## 模块分类说明

为了便于后续维护和查阅，按照功能类型对 `templates` 中的模块进行分类。

### 基础启动与信息类

| 模块名称     | 说明                                     |
| ------------ | ---------------------------------------- |
| hello      | 验证工程编译、下载、启动和串口输出流程。 |
| asm_hello  | 验证汇编工程启动和基础串口输出能力。     |
| archinfo   | 读取或显示系统架构相关信息。             |
| smoke_test | 快速检查系统基础功能是否正常。           |

### GPIO 与底层控制类

| 模块名称   | 说明                                           |
| ---------- | ---------------------------------------------- |
| gpio     | 验证通用 GPIO 输入输出功能。                   |
| asm_gpio | 验证汇编环境下的 GPIO 控制能力。               |
| rcu      | 验证复位控制、时钟控制或外设时钟使能相关功能。 |

### 总线与存储接口类

| 模块名称          | 说明                             |
| ----------------- | -------------------------------- |
| i2c_scan        | 扫描 I2C 总线设备地址。          |
| i2c_sgp30       | 通过 I2C 读取 SGP30 传感器数据。 |
| qspi            | 验证 QSPI 接口访问功能。         |
| spi_flash_test  | 验证 SPI Flash 的读写擦除功能。  |
| filesystem_test | 验证文件系统挂载和文件读写功能。 |



### 显示屏与图形应用类

| 模块名称           | 说明                                                 |
| ------------------ | ---------------------------------------------------- |
| spi_st7735       | 验证 ST7735 SPI 屏幕驱动和显示功能。                 |
| spi_st7735_clock | 在 ST7735 屏幕上显示时钟信息。                       |
| spi_st7735_dout  | 验证 ST7735 屏幕 SPI 数据输出。                      |
| st7789           | 验证 ST7789 屏幕驱动和显示功能。                     |
| minesweeper      | 扫雷小游戏应用示例，可用于验证显示、输入和应用逻辑。 |

### 系统外设类

| 模块名称 | 说明                             |
| -------- | -------------------------------- |
| timer  | 验证定时器、中断或周期计数功能。 |
| rtc    | 验证实时时钟功能。               |
| pwm    | 验证 PWM 输出功能。              |
| wdg    | 验证看门狗功能。                 |
| rng    | 验证随机数发生器功能。           |
| crc    | 验证 CRC 校验功能。              |

### 交互与综合测试类

| 模块名称      | 说明                                           |
| ------------- | ---------------------------------------------- |
| shell_test  | 验证 Shell 命令行交互功能。                    |
| ps2         | 验证 PS2 输入接口功能。                        |
| smoke_test  | 工程移植后进行快速综合验证。                   |
| minesweeper | 作为综合应用示例，可验证多个模块协同运行效果。 |
