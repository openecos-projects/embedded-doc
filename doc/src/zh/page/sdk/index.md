# ECOS Embedded SDK

## 版本简介

ECOS Embedded SDK 文档会随 SDK 版本演进。你可以在左侧栏顶部通过“显示版本目录”切换当前查看的版本导航，也可以通过本页链接进入不同版本的入口文档。通用文档内容不会因为该切换而发生变化。

## 发布版本

- SDK 2.0（推荐）: [进入文档](v2.0/gpio_v2.md)
- SDK 1.0: [进入文档](v1.0/gpio.md)

## 通用文档

- [环境准备](common/env.md)
- [快速上手](common/quickstart.md)

## 版本切换规则

- 在 API 文档页内切换版本时，会尝试跳转到同名 API 的对应版本（例如 gpio ↔ gpio_v2）。
- 目标版本未收录该 API 时，会跳转到 SDK 版本简介页。
- 在版本简介页或通用文档页切换时，当前页面不跳转，只更新左侧显示的版本目录。
