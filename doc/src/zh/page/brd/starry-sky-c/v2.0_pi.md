# 开发入门

## 项目概述

星空C2板卡主控芯片采用的是团队自研的 [retroSoC](https://github.com/retroSoC/retroSoC)，它是一个开源的 ASIC 定制框架，支持接入不同类型的开源 RISC-V 处理器核（自研或第三方）并进行流片。当前版本 retroSoC 内部集成的是国外著名开源 RISC-V 处理器核 [PicoRV32](https://github.com/YosysHQ/picorv32)，其完整了实现 RV32IMC 指令集，并可以被灵活配置成 RV32E、RV32I、RV32IC、RV32IM、RV32IMC 等多种组合形式。此外，retroSoC 还包含诸多开源外设模块，如 UART、SPI、I2C、GPIO、PWM 等，极大提升了板卡的展示与应用潜力。



!!! warning "注意事项"
    从整体上看，该板卡主控芯片在功能和性能上基本与 ST 公司的 F1 中低端系列相当，但是由于可能存在的软硬件错误，因此当前该板卡仅作为嵌入式原型搭建使用（如内部测试、参加比赛等），**务必不要将其应用到实际生产作业环境中**。



## 芯片介绍

### 硬件架构

| 特性            | 描述            |
| :-------------- | :-------------- |
| **处理器核心** | PicoRV32        |
| **指令集架构** | RISC-V rv32imac |
| **工作频率**   | 72MHz           |

### 内存系统

| 类型          | 容量  | 地址范围                    |
| :------------ | :---- | :-------------------------- |
| **片上**     | 128KB | `0x30000000` - `0x3001FFFF` |
| **外部**     | 8MB   | `0x40000000` - `0x407FFFFF` |
| **SPI Flash** | 16MB  | `0x00000000` - `0x00FFFFFF` |

### 外设模块

项目包含丰富的外设驱动模块：

#### 通信接口

*   **SYS_UART**：标准串口通信 
*   **HP_UART**：高性能串口通信 
*   **SPI**：标准 SPI 接口  
*   **I2C**：I2C 总线接口 
*   **PS2**：PS2 键盘/鼠标接口 

#### 存储与控制

*   **GPIO**：16 路通用输入输出 
*   **PWM**：4 路脉宽调制输出 
*   **Timer**：2 路定时器 
*   **Flash**：SPI Flash 控制器 
*   **PSRAM**：外部 PSRAM 控制器 

#### 其他

*   **RNG**：随机数生成器 

*   **ArchInfo**：系统架构信息 

	

## 板卡介绍

![board](../../../res/img/brd/starry-sky-c/v2.0/board1.png)

星空 C2 Pi 外设接口介绍如下：

*   **系统** ：
	*   1 个电源开关、红色电源指示灯  
	*   1 个TypeC 电源接口（同 SoC 的 UART 调试接口） 
	*   1 个拖拽式烧录器：HFP-LINK  
*   **SoC**：
	*   1 个 72M 有源晶振  
	*   1 个有源晶振焊盘位 
	*   1 个 UART 调试接口，使用 CP2102 进行协议转换 
	*   1 个复位按键 
	*   1 个 8MB 的 PSRAM  
	*   1 个 16MB 的 SPI Flash 
	*   1 个 PS2 接口 
	*   1 个 SPI 彩屏 
	*   2 组 IO 引脚扇出
