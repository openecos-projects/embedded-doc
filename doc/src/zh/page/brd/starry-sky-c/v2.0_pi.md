## 项目概述
## Project Overview

星空C2板卡主控芯片采用的是团队自研的 [retroSoC](https://github.com/retroSoC/retroSoC)，**它是一个开源的 ASIC 定制框架，支持接入不同类型的开源 RISC-V 处理器核（自研或第三方）并进行流片。**

The main control chip of the Starry Sky C2 board utilizes the team's self-developed retroSoC. It is an open-source ASIC customization framework that supports integrating different types of open-source RISC-V processor cores (self-developed or third-party) and facilitates tape-out.

当前版本 retroSoC 内部集成的是国外著名开源 RISC-V 处理器核 [PicoRV32](https://github.com/YosysHQ/picorv32)，其完整了实现 RV32IMC 指令集，并可以被灵活配置成 RV32E、RV32I、RV32IC、RV32IM、RV32IMC 等多种组合形式。

The current version of retroSoC integrates the renowned open-source RISC-V processor core PicoRV32. It fully implements the RV32IMC instruction set and can be flexibly configured into various combinations such as RV32E, RV32I, RV32IC, RV32IM, and RV32IMC.

此外，**retroSoC 还包含诸多开源外设模块，如 UART、SPI、I2C、GPIO、PWM 等，极大提升了板卡的展示与应用潜力。**

Furthermore, retroSoC also includes numerous open-source peripheral modules, such as UART, SPI, I2C, GPIO, PWM, etc., significantly enhancing the board's demonstration and application potential.

!!! warning "注意事项"
    从整体上看，该板卡主控芯片在功能和性能上基本与 ST 公司的 F1 中低端系列相当，但是由于可能存在的软硬件错误，因此当前该板卡仅作为嵌入式原型搭建使用（如内部测试、参加比赛等），**务必不要将其应用到实际生产作业环境中**。

    Overall, the functionality and performance of the board's main control chip are roughly equivalent to ST's F1 mid-to-low-end series. However, due to potential software or hardware errors, this board is currently intended for embedded prototyping use only (e.g., internal testing, participation in competitions). It must not be used in any actual production or operational environments.

## 板卡介绍
## Board Introduction

![board](../../../res/img/brd/starry-sky-c/v2.0/board1.png)

星空 C2 Pi 外设接口介绍如下：

Introduction to the Starry Sky C2 Pi peripheral interfaces:

*   **系统 System**：
    *   1 个电源开关、红色电源指示灯  1 Power switch, red power indicator LED
    *   1 个 TypeC 电源接口（同 SoC 的 UART 调试接口）  1 Type-C power interface (also serves as the SoC's UART debug interface)
    *   1 个拖拽式烧录器：HFP-LINK  1 Drag-and-drop programmer: HFP-LINK
*   **SoC**：
    *   1 个 72M 有源晶振  1 72MHz active crystal oscillator
    *   1 个有源晶振焊盘位  1 Active crystal oscillator pad position
    *   1 个 UART 调试接口，使用 CP2102 进行协议转换  1 UART debug interface, using CP2102 for protocol conversion
    *   1 个复位按键  1 Reset button
    *   1 个 8MB 的 PSRAM  1 8MB PSRAM
    *   1 个 16MB 的 SPI Flash  1 16MB SPI Flash
    *   1 个 PS2 接口  1 PS2 interface
    *   1 个 SPI 彩屏  1 SPI color screen
    *   2 组 IO 引脚扇出  2 Sets of fanned-out IO pins
